# Power BI — Анализ продаж аптек
## Скриншот отчета

![Главный дашборд](apteka.png)

Проект выполнен в Power BI на основе данных о продажах аптечной сети за 2015 год.

Данные загружены из Excel файлов. Файлы продаж (Аптека 1, 2 и 3) объединены в одну таблицу через подключение к папке. Позже в папку добавлены файлы за 3-й и 4-й кварталы. Справочники «География», «Курс» и «Товары» нужны для детализации продаж по отделам, пересчета валюты и описания товаров.

Таблица «План продаж» обработана в редакторе запросов: удалены пустые строки, развернуты столбцы, столбец «Период» преобразован из текста в дату. Для связи плана с товарами создана доп таблица «Товарные группы» с помощью функции VALUES.

Рассчитан столбец «Прибыль-факт» с использованием функции RELATED. Создана мера «План-факт» в процентном формате.

Отчет включает гистограмму с помесячной динамикой, линейчатую диаграмму по товарным группам, два датчика и срез по отделам. При добавлении новых файлов отчет обновляется кнопкой «Обновить». Готовый отчет опубликован.
